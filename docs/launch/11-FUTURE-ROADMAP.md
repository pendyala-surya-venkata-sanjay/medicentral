# Future Roadmap

## Near term (0–6 months)

- Redis Socket.io adapter for multi-instance API
- S3/Cloudinary production wiring (abstraction in place)
- FHIR read/export for interoperability
- Mobile-optimized ops views

## Mid term (6–18 months)

- HL7 ADT integration for admission/discharge
- Billing connectors (insurance eligibility)
- Advanced ML: triage suggestions with clinical review workflow
- Per-tenant SLA dashboards

## Long term

- Multi-region deployment
- Offline-first ward tablets
- National health ID integration (market-specific)

## What we will not compromise

- Workflow-first architecture
- Consent before cross-hospital data access
- Clear AI disclaimers on patient-facing features
